**history/+page.svelte:**

1. Add `import * as m from '$lib/paraglide/messages'` and `import { formatRelativeTime } from '$lib/utils/format'`.

2. **Remove the hand-rolled `relativeTime()` function** (or `formatRelativeTime` local function with English suffixes like "5m ago", "2h ago"). Replace all calls to it with the shared `formatRelativeTime()` from `$lib/utils/format`.

3. Replace all hardcoded strings:
   - Page title: `'Change History'` → `m.history_title()`
   - Revert button: `'Revert ({selectedCount})'` → `m.history_revert_button({ count: selectedCount })`
   - Filter labels: `'All items'` → `m.history_filter_all_items()`, `'All users'` → `m.history_filter_all_users()`, `'All sources'` → `m.history_filter_all_sources()`, `'User'` → `m.history_filter_source_user()`, `'MCP'` → `m.history_filter_source_mcp()`
   - Column headers: `'Time'` → `m.history_column_time()`, `'User'` → `m.history_column_user()`, `'Item'` → `m.history_column_item()`, `'Action'` → `m.history_column_action()`, `'Field'` → `m.history_column_field()`, `'Change'` → `m.history_column_change()`
   - Loading/empty: `'Loading history...'` → `m.history_loading()`, `'No change history yet.'` → `m.history_empty()`, `'No changes match...'` → `m.history_no_matches()`
   - Pagination: `'{selected} of {total} row(s) selected'` → `m.history_rows_selected({ selected, total })`, `'Previous'` → `m.history_previous()`, `'Next'` → `m.history_next()`
   - Aria-labels: `'Select all'` → `m.history_select_all()`, `'Select row'` → `m.history_select_row()`

4. For any `toLocaleDateString()` or `toLocaleString()` calls, add explicit locale: `toLocaleDateString(getLocale(), ...)`. Import `getLocale` from `$lib/paraglide/runtime`.

**HistoryActionBadge.svelte:**

1. Add `import * as m from '$lib/paraglide/messages'`.

2. Create a mapping function for action labels:
   ```
   const actionLabel = (action: string) => {
     const map: Record<string, () => string> = {
       create: m.history_action_create,
       update: m.history_action_update,
       delete: m.history_action_delete
     };
     return (map[action] ?? (() => action))();
   };
   ```

3. Replace `{action}` display with `{actionLabel(action)}`.

4. Replace `{viaMcp ? 'MCP' : 'User'}` with `{viaMcp ? m.history_source_mcp() : m.history_source_user()}`.
